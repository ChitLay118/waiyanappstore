Edit Source Code

This is a custom apps store. To edit the app list, download the source code and modify the apps array in the script.js file.

  1. Find the script.js file.
  2. Go to the EDITABLE AREA.
  3. Change the title, icon, category, purpose, and download_link for each app.
  4. Add a type: 'game', 'nogame', or 'edit'.

When you are done, upload the modified files back to your website.
